# Scroll Anchor

{% hint style="info" %}
CONCEPTUAL
{% endhint %}

Add scroll anchor adjustments, by recreating Webflow's adjusted smooth scroll.&#x20;

* Support `#hash` entry into the page
* Support navigation links
* Support fixed / sticky navs
* Support adjustable offsets by section
  * And globally&#x20;
